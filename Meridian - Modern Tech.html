Identity

Who it's for, what it feels like.

Meridian is for technical founders shipping infrastructure, dev-tools, and serious B2B SaaS. It should feel precise, quietly confident, and a little reserved — the opposite of pastel gradients and winking illustrations. The page is a terminal first and a brochure second. Where competitors decorate with drop-shadows and rounded cards, Meridian decorates with structure: grid rhythms, hairline rules, mono callouts, a corner crosshair that shows up once in every frame.

Read it like schematic documentation. Every line is load-bearing. Every decoration is a label.

Accent

A single accent. The user picks the hue; we bake the chroma (0.17) and lightness (0.78) so it always sits right on dark. Default: mint oklch(0.78 0.17 155).

Accent usage rules

  • Allowed: primary button fill, one inline word per hero, link hover underline, active tab marker, status dot, one inline marker per quote, the CTA arc, numeric eyebrow index.
  • Forbidden: body text color, card fills, section backgrounds, icons at size, gradients-as-decoration, more than one accent occurrence per viewport.
  • Budget: ≤ 6% of the visual weight of any viewport.

Elevation

Cards do not cast shadows. They are drawn with a 1px border at oklch(1 0 0 / 0.08). For emphasis, raise to /0.14 or introduce a soft internal gradient (ink-1 → ink). The only real shadow is on floating menus and the hero product card:

shadow-menu: 0 10px 24px -12px #000a, 0 2px 6px -2px #0006

Decorative layer

Hand-crafted, specific, never screaming.

  • Hairline grid. 96px × 96px, on hero only, masked radially so it fades at edges. 4% opacity.
  • Corner crosshairs. 14px tick marks in each corner of every framed region. They read as registration marks.
  • Numbered eyebrows. 01 — in accent, label in fg-3. Present on every section except hero.
  • The arc. Three concentric circle outlines in accent, bottom-right of the CTA block. Used exactly once per page.
  • Animated caret. A single _ blinking beside one hero word. The only motion on the page.

Rule of one: each decorative motif appears exactly in the place it earns. No sprinkles.

Components

Button, form, card, badge, iconography.

Buttons

  • Primary — accent fill, ink text, 10px 14px, 8px radius, 13.5px display weight 500. Hover brightens 6%.
  • Ghost — transparent, 1px line border, bone text. Hover: border line-strong, bg ink-1.
  • Quiet — text only in fg-2. Hover: fg-1. For secondary nav.
  • Small — 7px 11px, 12.5px label.

Form input

Bordered, filled dark: 1px line, ink-1 fill, 6px radius. Focus: border pulls to accent, no ring, no glow.

Cards

  • Flat — 1px line only. Default.
  • Raised — internal gradient ink-1 → ink, border line-strong. Use once per section max.

Badges

Pill, 10.5px mono uppercase, 4px 10px. Status badges: accent fill + ink text. Category badges: transparent + 1px line + bone-2.

Iconography

Lucide, 1.25px stroke (not the default 2px), 18px inside a 28px glyph container with 6px radius and a 1px line. Icon stroke is accent. Never use icons at 48px+; they become illustrations and break the voice.

Copy voice

How the words should sound when Lessgo generates them.

Short sentences. Engineer-to-engineer. No adjectives that don't earn their keep. Numbers are preferred over adjectives — "48ms" beats "blazing fast." No "unlock," no "empower," no "revolutionize." Prefer the indicative: Preview every change. Not: You'll be able to preview every change.

Cadence: one long sentence, one short. Set up, then land. Eyebrows are terse and numeric. The word "we" is rare; "you" is common.

Tokens — copyable

Everything the engineers need. Accent is the one user-facing knob; all other tokens are baked. Both CSS variables and a Tailwind-ready JSON object.

  --accent-ink:    oklch(0.18 0.02 155);
  --accent-dim:    oklch(0.78 0.17 155 / 0.14);

  /* neutrals */
  --ink:           oklch(0.09 0.008 260);
  --ink-1:         oklch(0.12 0.008 260);
  --ink-2:         oklch(0.16 0.008 260);
  --bone:          oklch(0.98 0.004 260);
  --bone-2:        oklch(0.72 0.006 260);
  --bone-3:        oklch(0.50 0.006 260);

  --line:          oklch(1 0 0 / 0.08);
  --line-strong:   oklch(1 0 0 / 0.14);
  --line-soft:     oklch(1 0 0 / 0.04);

  /* shadow — rare */
  --shadow-menu:   0 10px 24px -12px #000a, 0 2px 6px -2px #0006;

  /* type */
  --font-display:  "Inter Tight", system-ui, sans-serif;
  --font-body:     "Inter", system-ui, sans-serif;
