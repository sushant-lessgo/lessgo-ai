№ 01 Identity

A retail-first system the other nine can't hold: hierarchy in service of one question — should I buy this?

None of the previous nine systems handle commerce well. Folio is too editorial-chilly to push a buy button. Riot disrespects the price tag. Hearth is service-not-product. Mill is local-not-shippable. Pulse is creator-not-catalog. Stockroom is built around the verb every other system avoids: sell.

The product, photographed. Price visible. Add to bag. Everything else is decoration.

Center of gravity is product photography — every variant leads with imagery, not headlines. Image-slot driven so users drop in their own product shots. Price always visible up front, never behind a quote-form. Filters look like filters, badges like badges, and "low stock" never lies.

Five commitments

  • Product photo first. Each major beat leads with imagery slots, not type.
  • Price visible always. Cards show price up front; "request quote" is banned.
  • Restraint with urgency. "Low stock · 3 left" is allowed; flashing red countdowns are banned.
  • Filters that look like filters. Real faceted nav UI, not aspirational copy.
  • Three-color discipline. One brand color, one CTA color, one accent for sale. That's it.

№ 02 Type

Fraunces (variable serif) for display in Boutique; Inter Tight steps in for display in Catalog and Marketplace. Inter Tight body throughout. Mono for spec strings, badges, and SKUs.

  • Boutique — Fraunces 400 display with italic for emphasis. Editorial without being chilly.
  • Catalog — Inter Tight 600 display, Fraunces italic for emphasis only. Fast scanning, dense grid.
  • Marketplace — Inter Tight 700 display, italic Fraunces accent. Reads transactional.

№ 03 Color

Three-color discipline. One brand color (deep, used on announcement bar + footer + ink-buttons). One CTA color (warm and singular, used only on primary buttons). One sale color (sparingly, badges only).

№ 04 Decoration

Five permitted motifs. Everything else is in service of the photo.

  • Announcement bar. Top strip, brand-color, three benefits separated by dots. Frees the hero from "free shipping" copy.
  • Hero stamp. A small rotated badge over the hero photo (e.g. "New / Spring"). Used once per page.
  • Product badges. "New", "Sale", "Low stock" — small, rectangular, one per card max.
  • Quick actions on hover. Card-bottom row of buttons (Add · Quick view) that slide in on hover.
  • Star rating with letter-fill. SVG-free star bars driven by background-clip.

What's banned: countdown timers, flashing "sale ends in" banners, "X people viewing this," fake-review carousels, gradient buttons, drop-shadow on every card, fake "members get more" gates.

Tokens, copy & paste.

Configurable axes are [data-palette] (9 hues) and [data-variant] (Boutique · Catalog · Marketplace). Display family swaps from Fraunces to Inter Tight on Catalog/Marketplace; product grid columns and card chrome change per variant.

:root tokens

/* Surface */
--paper:    oklch(0.985 0.004 80);     /* canvas */
--paper-1:  oklch(0.965 0.006 80);     /* second */
--ink:      oklch(0.18 0.012 60);       /* type */
--ink-2:    oklch(0.36 0.012 60);       /* body */
--rule:     oklch(0.88 0.008 60);       /* hairline */

/* Brand — announcement bar, ink buttons, footer */
--brand:    oklch(0.18 0.012 60);
--brand-on: oklch(0.985 0.004 80);

/* CTA — primary "Add to Bag". Never decoration. */
--cta:      oklch(0.32 0.06 30);
--cta-deep: oklch(0.22 0.06 30);

/* Sale — badge color only */
--sale:     oklch(0.55 0.18 28);

/* Type */
--font-display: "Fraunces", "Iowan Old Style", serif;
--font-body:    "Inter Tight", system-ui, sans-serif;
--font-mono:    "JetBrains Mono", monospace;

/* Radius — soft on photo, square on chrome */
--r-sm: 4px;  --r-md: 8px;
--r-lg: 14px; --r-pill: 999px;

palette overrides ([data-palette])

html[data-palette="ink"]      { --brand: oklch(0.18 0.012 60);  --cta: oklch(0.32 0.06 30); }   /* default */
html[data-palette="cream"]    { --brand: oklch(0.22 0.020 50);  --cta: oklch(0.42 0.13 35); }
html[data-palette="olive"]    { --brand: oklch(0.36 0.06 115);  --cta: oklch(0.40 0.13 30); }
html[data-palette="rose"]     { --brand: oklch(0.36 0.10 20);   --cta: oklch(0.42 0.16 15); }
html[data-palette="cobalt"]   { --brand: oklch(0.32 0.10 245);  --cta: oklch(0.50 0.18 30); }
html[data-palette="sand"]     { --brand: oklch(0.40 0.06 70);   --cta: oklch(0.45 0.13 30); }
html[data-palette="forest"]   { --brand: oklch(0.30 0.08 155);  --cta: oklch(0.45 0.13 30); }
html[data-palette="burgundy"] { --brand: oklch(0.30 0.10 18);   --cta: oklch(0.45 0.18 25); }
html[data-palette="slate"]    { --brand: oklch(0.32 0.018 240); --cta: oklch(0.42 0.13 30); }

variant overrides ([data-variant])

/* Boutique (default) — Fraunces display, 4-col curated grid, full-bleed hero. */

/* Catalog — sans display, 5-col dense grid, larger filter UI. */
html[data-variant="catalog"] .products { grid-template-columns: repeat(5, 1fr); }
html[data-variant="catalog"] .sec-title { font-family: var(--font-body); font-weight: 600; }

/* Marketplace — square photos, card-on-card, ships-in pill per item. */
html[data-variant="marketplace"] .product .photo { aspect-ratio: 1 / 1; }
html[data-variant="marketplace"] .product {
  background: var(--paper);
  border: 1px solid var(--rule);
  border-radius: var(--r-md);
  padding: 12px;
}
