№ 01 Identity

Hand-set, not hand-drawn. Letterpress vocabulary. One ink, one paper.

Mill is the system for the place you walk past every Tuesday. The neighbourhood bakery, the tiny restaurant with eight tables, the studio where someone is still throwing pots at 7pm. It looks made — by a person, on equipment that has a serial number — but it is never twee. There is no chalk-script. There are no rope-borders. There are no badges that say "EST. 2024" in a fake-vintage typeface.

Instead Mill borrows from the things actual makers actually use. Wood-type posters. Letterpress menus. Chalkboards (real ones). Butcher-paper price tags. Ledger-bound order books. Stamped boxes. The vocabulary is concrete and the typography is correct.

"Apricots. From Helen, in Sebastopol. Tuesday only." — not "premium artisanal stone-fruit."

The system enforces specificity in the copy slots. Every product has a person, a place, a day. Every price is a real number. Every "About" page is shorter than its menu.

№ 02 Type

Wood-type display. Italic serif for the editorial moment. Mono for tickets and stamps.

Variants

  • Letterpress — Oswald (wood-type stand-in) for display, Source Serif 4 for body, Fraunces italic for editorial. The default — small hotels, restaurants, wineries.
  • Market — same families, inverted to chalk-on-slate paper. Adds two-color tickets and hand-pricing.
  • Workshop — blueprint background, more JetBrains Mono for source-tags and dimensional callouts. Ceramic, leather, wood studios.

№ 04 Decoration

Six permitted motifs. They were used because they were useful, then they became style.

  • Stars and asterisks (✶ ⁂) — section breaks, nav indicators, list markers. Always Fraunces italic, always at one weight below the body.
  • Double rules — 3px double-line at section breaks and chapter heads, the printer's way of saying "this is important."
  • Stamps and seals — outline circle with a hand-set monogram. One per page, in the masthead.
  • Tickets & price tags — dashed-edged cards with a tag, a name, a where, a price. The composing-stick of the system.
  • Hand-numbered editions — "№ 14 of 200" in the corner of any limited thing. Builds inventory honesty.
  • Paper grain — radial-noise overlay at low opacity. The page should feel printed, never sterile.

What's banned: chalk-script fonts, rope borders, burlap textures, photo filters that fake age, "EST." badges, leaf flourishes, anything painted on a wooden sign.

Tokens, copy & paste.

Mill is a single-ink-on-single-paper system. The two configurable axes are --ink + --paper (palette, 9 options) and [data-variant] (Letterpress · Market · Workshop). Display family swaps with variant; body stays Source Serif 4.

:root tokens

/* Single-ink palette — the only color knob */
--ink:        oklch(0.34 0.10 40);     /* terracotta */
--ink-deep:   oklch(0.22 0.10 40);     /* hover / press */
--ink-soft:   oklch(0.50 0.09 40);     /* captions */
--ink-pale:   oklch(0.78 0.05 40);     /* hairlines */
--paper:      oklch(0.96 0.018 85);    /* cream */
--paper-1:    oklch(0.93 0.022 85);
--paper-2:    oklch(0.89 0.026 85);

/* Type */
--font-display: "Fraunces", serif;          /* italic editorial moments */
--font-wood:    "Oswald", sans-serif;       /* condensed wood-type for headlines */
--font-body:    "Source Serif 4", serif;
--font-mono:    "JetBrains Mono", monospace;

/* Radius — none. Mill is square stock. */
--r-sm: 0; --r-md: 0; --r-lg: 0;

palette overrides ([data-palette])

html[data-palette="clay"]   { --ink: oklch(0.34 0.10 40);  --paper: oklch(0.96 0.018 85); } /* default */
html[data-palette="olive"]  { --ink: oklch(0.36 0.07 110); --paper: oklch(0.96 0.014 90); }
html[data-palette="ochre"]  { --ink: oklch(0.42 0.10 70);  --paper: oklch(0.96 0.020 85); }
html[data-palette="rust"]   { --ink: oklch(0.30 0.12 30);  --paper: oklch(0.93 0.022 70); } /* on kraft */
html[data-palette="indigo"] { --ink: oklch(0.28 0.08 250); --paper: oklch(0.96 0.012 90); }
html[data-palette="mill"]   { --ink: oklch(0.22 0.012 60); --paper: oklch(0.96 0.010 90); }
html[data-palette="cellar"] { --ink: oklch(0.28 0.11 20);  --paper: oklch(0.95 0.018 85); }
html[data-palette="linen"]  { --ink: oklch(0.36 0.07 145); --paper: oklch(0.95 0.014 90); }
html[data-palette="brick"]  { --ink: oklch(0.36 0.13 28);  --paper: oklch(0.93 0.024 80); }

variant overrides ([data-variant])

/* Letterpress (default) — wood-type headlines, single ink, cream paper. */

/* Market — chalkboard. Inverted: ink becomes paper, paper becomes slate. */
html[data-variant="market"] {
  --paper: oklch(0.18 0.012 60);     /* slate */
  --ink:   oklch(0.96 0.018 85);     /* chalk */
}

/* Workshop — blueprint. Cool paper, faint grid background. */
html[data-variant="workshop"] {
  --paper: oklch(0.95 0.012 220);
  --ink:   oklch(0.30 0.08 240);
}
html[data-variant="workshop"] body {
  background-image: linear-gradient(to right, oklch(0.30 0.08 240 / 0.04) 1px, transparent 1px),
                     linear-gradient(to bottom, oklch(0.30 0.08 240 / 0.04) 1px, transparent 1px);
  background-size: 32px 32px;
}
