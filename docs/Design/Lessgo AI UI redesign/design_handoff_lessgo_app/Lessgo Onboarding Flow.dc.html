STEP 04 Show the plan — before building

Answers three questions in ~10 seconds: what pages, what’s on each, what each asks visitors to do. Real photos appear where they’ll go — the difference between “a plan” and “MY site.” Plain words only, never Hero/CTA. This is a working draft, not an FYI — she can rename pages, reorder them, change any page’s goal, swap which work leads, or add/remove a page. Doing nothing is fine too: approving starts the build, and everything stays editable after.
