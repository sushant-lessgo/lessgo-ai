Identity

Who it's for, what it feels like.

Lumen is for a photographer whose portfolio is the pitch — corporate headshots, personal branding, product, events, portraits. The buyer is a marketing lead, founder, or office manager who wants to look credible to clients, hires and press. The page must feel premium and composed, never wedding-influencer flashy: large imagery, generous air, restrained type, and a clear, repeated path to enquiry. Trust comes from the work and from specificity — named packages, real prices, plain deliverables.

Hang the work like a gallery, not a brochure. Every image gets a caption; the UI never competes with the photograph.

Typography

An editorial serif for voice; a clean sans for trust; a mono for the catalogue.

Pairing

  • Spectral — display & headings, often italic for the accent word. Editorial, premium, calm.
  • Inter — body & UI. Neutral and legible; holds longer Dutch strings without fuss.
  • JetBrains Mono — the catalogue voice: figure numbers, category tags, prices' unit, eyebrows, labels.

Usage rules

  • Ink — primary structure: headings, the dark CTA button fill, body text.
  • Espresso — dark sections (process band, footer, lightbox scrim) so photos can sit on a calm field.
  • Brass — used sparingly: the italic accent word, figure numbers, the active language, primary booking CTA, hover underlines, frame-lines. Never a large fill.
  • Forbidden: brass as a background field, gradients-as-decoration, more than one accent hue, drop-shadows on photos (the work needs no glow).

Corners are near-square — premium editorial reads sharp, not bubbly. Elevation is hairline-first; cards lift only on hover. Photos carry no shadow, no glow, no rounded corners beyond the 2px frame.

Signature: editorial caption + mixed-ratio gallery

The motif that makes a wall of images read like a curated show.

Every image carries a .fig caption — a mono figure number, the category, and the aspect ratio (FIG. 03 — CORPORATE — 3:2). The gallery natively mixes landscape 3:2 (commercial work) and portrait 4:5 (personal work) on the same grid without awkward crops. Each category cover opens a lightbox — full-frame viewer with prev/next, a counter, dot index, and keyboard nav (←/→/Esc), the scrim in espresso so the photograph is the only bright thing on screen.

Other components

  • Service card — name + Dutch subtitle + price + pitch + a ticked "what you get" list + enquiry link.
  • Buttons — fill (ink), brass (booking), line (outline), wa (WhatsApp), ghost-d (on dark).
  • Frame-line — a thin brass inset rectangle on hero/about portraits, like a viewfinder.
  • Floating CTA — persistent WhatsApp + book-a-call, bottom-right, collapses to icons on mobile.

Bilingual (EN / NL)

Built to switch language without breaking layout.

A mono EN · NL toggle in the nav flips every translatable node via data-en / data-nl attributes (and data-en-ph / data-nl-ph for inputs); the choice persists in localStorage and sets <html lang>. Service cards also show a permanent Dutch subtitle under the English title, so the page reads bilingual even before you toggle. Leave headline measures loose — Dutch runs ~15% longer than English.

Copy voice

Confident, concrete, client-first. No adjectives the work can prove itself.

Lead with what the client gets from the photograph — credibility with clients, hires and press — then the practical detail. Name deliverables plainly ("5 retouched finals · 48h delivery"). Prices are shown, ex. btw, never hidden behind "request pricing". Skip "capture your story", "passionate", "magical moments". CTAs offer a conversation or a booking, repeated naturally down the page.

"LinkedIn-ready headshots, delivered in 48 hours" beats "timeless portraits that capture the real you" every time.

Page structure

The ordered section system this template ships with.

  • Nav — slim, sticky; links + EN/NL toggle + Request-a-quote.
  • Hero — positioning headline + subtitle + dual CTA, with a large portrait and a viewfinder frame-line.
  • Client-type strip — who she shoots for (law firms · consultancies · startups · agencies).
  • Services — 4 priced package cards with Dutch subtitle + deliverables.
  • Process — dark band, 3 steps (enquire → shoot day → delivery).
  • Portfolio — Commercial (3:2) & Personal (4:5) groups; each category opens a lightbox.
  • About — photographer portrait + intro + enquiry.
  • Contact — form + direct details + WhatsApp + book-a-call.
  • Footer + persistent floating CTA.

Configure for a client

The knobs to turn when reusing this template. Everything else is baked.

brand
Wordmark in nav/footer, page <title>, positioning line.
--brass
One hue knob. Shift the accent hue in :root; keep lightness/chroma so contrast holds.
contact
Phone · WhatsApp · email · city (Den Haag · +31 6 2949 3643 · [email protected]) + booking-call URL.
services
Package cards — names, Dutch subtitles, prices, pitches, deliverables. Add/remove cards.
portfolio
Categories & lightbox sets — group, ratio (3:2 / 4:5), image count per category.
imagery
Replace every .ph with a real photo. The fig tag names each slot.
language
data-en / data-nl strings; default language & toggle order.
sale model
Prices shown by default. Swap to "enquire" per card if a client prefers no public pricing.

Tokens — copyable

Everything an engineer needs to stand the template up. The one brand hue (--brass) is the user-facing knob; the rest is baked. CSS variables plus a Tailwind-ready snippet.

CSS variables

/* lumen.css — light · warm gallery · one brass knob */
:root {
  /* surfaces — warm gallery white */
  --paper:    oklch(0.977 0.006 70);
  --paper-2:  oklch(0.958 0.008 68);
  --paper-3:  oklch(0.936 0.010 66);

  /* ink — warm near-black */
  --ink:      oklch(0.235 0.010 60);
  --ink-2:    oklch(0.435 0.012 62);
  --ink-3:    oklch(0.595 0.012 64);

  /* espresso — dark surfaces */
  --esp:      oklch(0.258 0.012 56);
  --esp-d:    oklch(0.205 0.010 52);

  /* brass — the one accent (hue knob) */
  --brass:    oklch(0.745 0.066 72);
  --brass-d:  oklch(0.565 0.072 64);   /* legible on paper */
  --brass-l:  oklch(0.815 0.050 74);   /* on espresso */

  /* lines */
  --line:     oklch(0.905 0.008 68);
  --line-2:   oklch(0.842 0.012 66);
  --line-dk:  oklch(0.400 0.012 56);  /* on espresso */

  /* type */
  --font-display: "Spectral", Georgia, serif;
  --font-body:    "Inter", system-ui, sans-serif;
  --font-mono:    "JetBrains Mono", ui-monospace, monospace;

  /* rhythm */
  --pad-y:    clamp(76px, 9vw, 132px);
  --pad-x:    clamp(20px, 5vw, 68px);
  --r:        2px;   --r-lg: 3px;
  --maxw:     1240px;
}
