№ 01Identity

Two accents. They fight. The fight is the design.

Riot is the system that disrespects whitespace on purpose. Where Folio holds 160px of section padding sacred, Riot pushes the type to the edge of the viewport and lets letters touch. Where Hearth uses a single warm accent at 8% of the viewport, Riot uses two high-chroma accents at ~25% combined and lets them argue.

This is the system for buyers who want the page to feel like a poster pinned to a wall. Agencies whose hero is their own work. Fashion brands that don't care if you can read it on first glance. SaaS that wants to sound less like a product and more like a manifesto.

If your buyer asks "is this too much?" — they are not your buyer.

The accent budget is inverted: ~3% in Folio, ~25% in Riot. The rule is no longer "hold accent at minimum" — it's "two accents must always appear in the same viewport, and never be the same hue." If a viewport has only one accent, the design has failed.

№ 02Type

Display fills the page. Body holds back. Italics in Instrument Serif provide the only softness.

Variants

  • Brutalist (default) — Bricolage Grotesque 800 for display, all-caps. Geometric, hard, technical. Body in Space Grotesk.
  • Maximal — same display family but italic Instrument Serif used at title sizes too. Mixed-font tension is the whole point.
  • Riso — Instrument Serif italic at display sizes. Buttons go serif italic. The zine variant.

№ 04Decoration

Six permitted motifs. Use as many as you like.

  • Marquees. Black ticker band running across full bleed, oversized, with serif italics breaking the all-caps.
  • Stickers. Round, rotated badges with hard stamp shadows. Always at the corner of a hero image or section.
  • Stamp shadows. Hard offset shadows in --ink (8 8 0). Buttons and cards have them. Hover lifts to 10 10.
  • Marker eyebrows. Two-tone bordered pills. Number on the left in accent, label on the right.
  • Italic serif pops. One italic word per heading, in Instrument Serif, contrasting tone block. The shape contrast carries the brand.
  • Big-word footers. The brand name set at 280px in the footer, full-bleed. Always.

No drop shadows below 8px. No gradients (except in Maximal variant body wash). No rounded corners except round stickers and pill stamps.

№ 05Components

All components carry the line-weight system. Borders are 2px on small, 4px on full-bleed dividers. Always full-strength --ink, never softened.

Buttons

  • Primary — --a2 fill, ink border, 8px stamp shadow. Hover translates -2px (shadow grows). Active translates +4px (shadow shrinks).
  • Ghost — bg fill, ink border, stamp shadow. Hover fills with --a1.
  • Quiet — text only, 2px underline. The only un-stamped action.

Cards

Every card is a stamped tile — hard borders, hard shadows. Featured cards rotate slightly (-2deg) or translate up. Three-card sections use three different fills (bg / a1 / a2) — never one fill for all three.

Forms

Inputs have hard 2px borders, no rounding, no inner shadow. Focus thickens border to 4px. Submit button is always a stamped Primary.

TOKENS, copy & paste.

The full token surface for the renderer. The user-configurable axes are --a1 + --a2 (palette, 9 dual-accent options) and [data-variant] (Brutalist · Maximal · Riso). Every component reads from both accents — never one in isolation.

:root tokens

/* Dual accent — both required, never use one alone */
--a1:        oklch(0.85 0.21 130);   /* primary — usually lighter */
--a1-deep:   oklch(0.62 0.21 130);
--a1-ink:    oklch(0.10 0.02 130);
--a2:        oklch(0.62 0.27 0);     /* secondary — usually saturated */
--a2-deep:   oklch(0.46 0.27 0);
--a2-ink:    oklch(0.99 0.01 0);

/* Surface */
--bg:        oklch(0.99 0.005 90);
--bg-deep:   oklch(0.10 0.015 240);
--ink:       oklch(0.12 0.015 240);

/* Lines + shadow — full strength always */
--line:           var(--ink);
--line-w:         2px;
--line-thick:     4px;
--shadow-stamp:    8px 8px 0 var(--ink);

/* Type */
--font-display: "Bricolage Grotesque", "Space Grotesk", sans-serif;
--font-sans:    "Space Grotesk", system-ui, sans-serif;
--font-serif:   "Instrument Serif", serif;       /* italic accents only */
--font-mono:    "JetBrains Mono", monospace;

/* Radius — Riot is square */
--r-sm: 0; --r-md: 0; --r-lg: 0; --r-xl: 0;
--r-pill: 999px;   /* only for round stickers */

/* Section rhythm */
--sec-pad-y:  120px;
--sec-pad-x:  clamp(20px, 4vw, 64px);

palette overrides ([data-palette])

html[data-palette="acid"]        { --a1: oklch(0.85 0.21 130); --a2: oklch(0.62 0.27 0); }   /* lime + magenta (default) */
html[data-palette="punch"]       { --a1: oklch(0.92 0.20 100); --a2: oklch(0.58 0.24 25); }  /* sun + cherry */
html[data-palette="volt"]        { --a1: oklch(0.74 0.18 230); --a2: oklch(0.72 0.20 50); }  /* electric + tangerine */
html[data-palette="toxic"]       { --a1: oklch(0.88 0.22 125); --a2: oklch(0.50 0.22 290); } /* lime + violet */
html[data-palette="ember"]       { --a1: oklch(0.74 0.20 60);  --a2: oklch(0.42 0.22 260); } /* orange + cobalt */
html[data-palette="bubblegum"]   { --a1: oklch(0.74 0.20 350); --a2: oklch(0.62 0.14 195); } /* pink + teal */
html[data-palette="mono"]        { --a1: oklch(0.92 0.20 100); --a2: oklch(0.12 0.015 240); } /* sun + black */
html[data-palette="highlighter"] { --a1: oklch(0.96 0.22 110); --a2: oklch(0.62 0.27 0); }   /* neon yellow + magenta */
html[data-palette="risograph"]   { --a1: oklch(0.62 0.16 250); --a2: oklch(0.70 0.22 350); } /* cornflower + fluo pink */

variant overrides ([data-variant])

/* Brutalist (default) — black + accents, geometric, hard. */

/* Maximal — full color tiles + radial body wash */
html[data-variant="maximalist"] body {
  background: radial-gradient(circle at 95% 8%, var(--a1) 0, transparent 14%),
              radial-gradient(circle at 4% 50%, var(--a2) 0, transparent 12%),
              var(--bg);
}

/* Riso — halftone fills + serif italic display */
html[data-variant="riso"] body {
  background-image: radial-gradient(circle at 1px 1px,
    oklch(0.20 0.02 240 / 0.15) 1px, transparent 1.4px);
  background-size: 4px 4px;
}
html[data-variant="riso"] .section-title,
html[data-variant="riso"] .doc-title,
html[data-variant="riso"] .footer-bigword {
  font-family: var(--font-serif);
  font-style: italic;
  text-transform: none;
